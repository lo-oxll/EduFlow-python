# نظام ترقية الطلاب والحفاظ على السجل الأكاديمي

## 📋 نظرة عامة

تم تطبيق نظام متكامل لترقية الطلاب يحافظ على السجل الأكاديمي الكامل للطالب عبر جميع السنوات الدراسية.

---

## ✨ الميزات الرئيسية

### 1. **ترقية الطالب** 
عند ترقية طالب:
- ✅ يتم تحديث مستوى الطالب الدراسي فقط (مثال: من "الأول الابتدائي" إلى "الثاني الابتدائي")
- ✅ يتم إنشاء سجل جديد فارغ للسنة الدراسية الحالية
- ✅ تبقى جميع درجات السنوات السابقة محفوظة في قاعدة البيانات
- ✅ لا يتم نقل أو تعديل أو حذف أي درجات عند الترقية

### 2. **الحفظ التلقائي للسجلات**
- ✅ كل سنة دراسية لها سجل منفصل في جدول `student_grades`
- ✅ يتم ربط كل سجل بـ `academic_year_id` الخاص بتلك السنة
- ✅ السجلات محفوظة بشكل دائم ولا يمكن تعديلها بالخطأ

### 3. **زر عرض السجل الأكاديمي**
- ✅ زر "السجل" متاح في قائمة كل طالب
- ✅ يعرض جميع السنوات الدراسية التي درس فيها الطالب
- ✅ مقسم إلى تبويبين:
  - **الدرجات**: يظهر جميع المواد والدرجات لكل سنة
  - **الحضور**: يظهر سجل الحضور والغياب لكل سنة

## 📊 هيكل قاعدة البيانات

### جدول الطلاب (`students`)
```sql
CREATE TABLE students (
    id INT PRIMARY KEY,
    full_name VARCHAR(255),
    student_code VARCHAR(50),
    grade VARCHAR(100),  -- هذا الحقل هو الذي يتغير عند الترقية
    school_id INT,
    created_at TIMESTAMP,
    updated_at TIMESTAMP
);
```

### جدول الدرجات (`student_grades`)
```sql
CREATE TABLE student_grades (
    id INT PRIMARY KEY,
    student_id INT,
    academic_year_id INT,  -- يربط الدرجة بسنة دراسية محددة
    subject_name VARCHAR(100),
    month1 DECIMAL(5,2),
    month2 DECIMAL(5,2),
    midterm DECIMAL(5,2),
    month3 DECIMAL(5,2),
    month4 DECIMAL(5,2),
    final DECIMAL(5,2)
);
```

#### قبل الترقية (الصف الأول - 2023-2024):

## 🔍 الاستعلامات المفيدة

### عرض جميع طلاب صف معين مع سجلاتهم
```sql
SELECT s.full_name, s.grade, s.student_code,
       sg.academic_year_id, sg.subject_name, sg.final
FROM students s
LEFT JOIN student_grades sg ON s.id = sg.student_id
WHERE s.grade = 'الثاني الابتدائي'
ORDER BY s.full_name, sg.subject_name;
```

### عرض درجات طالب عبر جميع السنوات
```sql
SELECT say.name as academic_year, 
       sg.subject_name, 
       sg.final as final_grade
FROM student_grades sg
JOIN system_academic_years say ON sg.academic_year_id = say.id
WHERE sg.student_id = 123
ORDER BY say.start_year DESC, sg.subject_name;
```

### إحصائيات الطالب
```sql
SELECT say.name, 
       COUNT(DISTINCT sg.subject_name) as subjects_count,
       AVG(sg.final) as average_grade
FROM student_grades sg
JOIN system_academic_years say ON sg.academic_year_id = say.id
WHERE sg.student_id = 123
GROUP BY say.id, say.name
ORDER BY say.start_year DESC;
```
